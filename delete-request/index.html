1. 삭제 가능한 데이터 유형

약국나우는 사용자 계정을 만들지 않으며, 익명 디바이스 식별자(해시)로만 작성한 콘텐츠를 식별합니다. 다음 데이터를 삭제 요청할 수 있습니다.

  • 약국 리뷰: 별점(친절·설명·대기), 코멘트 텍스트
  • 리뷰 신고 이력: 신고 사유, 신고 일시
  • 영업정보 오류 신고: 신고 유형, 메모
  • 복약 공유 스냅샷: "가족과 공유"로 생성한 30일짜리 링크 데이터
  • 익명 디바이스 ID(해시): 위 데이터에 연결된 식별자

※ 위치 정보, 처방전 사진, 복약 알림 설정 등은 처음부터 서버에 전송되지 않으며 기기에만 저장됩니다. 앱을 삭제하거나 앱 내 "전체 삭제"로 즉시 파기됩니다.

2. 삭제 요청 방법

방법 1: 앱 내에서 직접 삭제 (가장 빠름)

STEP 1 약국나우 앱 실행 → 하단 설정 탭으로 이동
STEP 2 "내가 작성한 리뷰" 또는 "내가 신고한 항목" 선택
STEP 3 개별 항목 우측 "삭제" 버튼 → 즉시 서버에서 영구 삭제
전체 삭제 설정 → "내 데이터 전체 삭제" → 본 기기에서 작성한 모든 서버 데이터 일괄 삭제

방법 2: 이메일로 삭제 요청

아래 양식으로 pharmacynow.kr@gmail.com으로 보내주세요.

  1. 제목: [데이터 삭제 요청] 약국나우
  2. 본문:
    • 삭제할 데이터 유형 (예: 약국 리뷰)
    • 식별 정보 중 하나 이상:
      • 리뷰 작성한 약국 이름·주소
      • 리뷰 작성 대략적인 날짜
      • 또는 앱 설정 화면에 표시된 "내 디바이스 ID" 끝자리 8자
    • 삭제 범위: 일부 / 전체

처리 기간: 영업일 기준 7일 이내 처리 후 완료 회신 드립니다.

1. Data Eligible for Deletion

PharmacyNow has no user accounts; user-generated content is identified only by an anonymous device hash. You may request deletion of:

  • Pharmacy reviews: ratings (friendliness, explanation, wait) and comment text
  • Review reports: report reason and timestamp
  • Data error reports: report kind and note
  • Medication share snapshots: 30-day shared links created via "Share with family"
  • Anonymous device hash: the identifier linking the above data

Note: Location, prescription photos, and medication reminder settings are never sent to our servers in the first place — they live only on your device and are erased when you uninstall the app or use "Delete all" in settings.

2. How to Request Deletion

Option 1: In-app deletion (fastest)

STEP 1 Open PharmacyNow → bottom Settings tab
STEP 2 Tap "My Reviews" or "My Reports"
STEP 3 Tap "Delete" next to each item — permanently removed from the server
Delete all Settings → "Delete all my data" erases every server-side record created from this device at once

Option 2: Request by email

Email pharmacynow.kr@gmail.com with the following:

  1. Subject: [Data Deletion Request] PharmacyNow
  2. Body:
    • Type of data to delete (e.g., pharmacy reviews)
    • At least one identifier:
      • Pharmacy name/address you reviewed
      • Approximate date of the review
      • Or the last 8 characters of the "My Device ID" shown in app settings
    • Scope: partial / all

Turnaround: We reply with a completion confirmation within 7 business days.

3. Retention Policy After Deletion

Data type Retention after deletion Reason
Review content Immediately and permanently deleted —
Review/error report records Immediately and permanently deleted —
Share snapshots Immediately and permanently deleted —
Database automatic backups Up to 30 days Disaster recovery; backups are also purged after 30 days

PharmacyNow does not collect any personal data subject to legal retention obligations (no e-commerce, no financial transactions). No additional data is retained after deletion.

4. Account Deletion

PharmacyNow has no sign-up or login. Users are identified only by an anonymous device hash, so there is no separate "account deletion" — once your content and identifier are removed via the steps above, no information about you remains in our system.
